# PRD — TAAS 결빙(블랙아이스) 교통사고 분석 웹사이트

## 1. 한 줄 정의
TAAS(교통사고분석시스템)의 **결빙 교통사고 다발지역** 공공데이터를 가공해, 전국 위험지역을 지도·차트·표로 보여주는 무료 분석 웹사이트.

## 2. 배경 / 문제
- 결빙(블랙아이스) 사고는 계절성·치명성이 높은데, 일반인이 "어디가 위험한지"를 한눈에 볼 곳이 없다.
- TAAS 원본 데이터는 분석·시각화가 안 된 raw 형태라 접근성이 낮다.

## 3. 타깃 사용자
- 겨울철 운전자 (내 동선의 위험지역 확인)
- 데이터/정책 관심자, 포트폴리오 열람자

## 4. MVP (v1) 범위 — "되는지" 검증
**핵심 데이터:** 도로교통공단 결빙(노면) 교통사고 다발지역 (위경도 포함)

**화면 3개:**
- [ ] 🗺️ 전국 결빙사고 다발지역 지도 (마커/히트맵, 클릭 시 사고건수·사상자 팝업)
- [ ] 📊 지역별 순위 차트 (시도별 다발지역 수 / 사상자 Top 10)
- [ ] 📋 원본 데이터 테이블 (검색·필터)

**비범위 (v1에서 안 함):**
- 자동 데이터 수집(GitHub Actions) — 결빙 데이터는 연 단위 갱신이라 불필요
- DB(Supabase) — 정적 CSV로 충분
- 로그인/회원/다른 사고유형

## 5. 기술 스택
| 영역 | 선택 |
|------|------|
| 언어 | Python 3.10 |
| UI | Streamlit |
| 지도 | pydeck |
| 차트 | plotly |
| 전처리 | pandas |
| 저장 | repo 내 CSV/GeoJSON (DB 없음) |
| 배포 | Streamlit Community Cloud (GitHub 연동, 무료) |

## 6. 데이터 출처 / 라이선스
- 출처: 도로교통공단 TAAS / 공공데이터포털(data.go.kr)
- 공공데이터 이용약관 준수 — 사이트 푸터에 출처표시 명기

## 7. 성공 기준
- v1: 샘플 1건이 지도+차트+표로 실제 렌더되는 것을 눈으로 확인 → 실데이터 연결 → Streamlit Cloud 배포 성공
